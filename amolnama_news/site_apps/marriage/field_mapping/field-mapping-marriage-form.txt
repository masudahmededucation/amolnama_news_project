================================================================================
 FIELD MAPPING: Register Marriage / Nikahnama (Form 'ঘ')
================================================================================
 Template: marriage/marriage-form.html
 URL:      /marriage/register-marriage/  (also /marriage/nikahnama/ — hidden card)
 View:     marriage.views.register_marriage / marriage.views.marriage_form
 Storage:  Client-side only (no backend DB) — PDF/print output
================================================================================

NOTE: This form is entirely client-side. No data is saved to any database.
All fields are rendered into an A4 paginated preview for print/PDF export.

JS files:
  - marriage/assets/js/components/marriage-sigpad.js          (signature pad — shared)
  - marriage/assets/js/pages/marriage-form-stepper.js         (multi-step stepper)
  - marriage/assets/js/pages/marriage-form-preview.js         (A4 preview builder)

CSS files:
  - marriage/assets/css/components/marriage-sigpad.css        (signature pad — shared)
  - marriage/assets/css/pages/marriage-form.css               (form + preview styles)
