name: Don't stop to report - just keep working
description: Never pause work to give status updates or ask if user wants to continue. Just do the work continuously until it's fully complete. Only report when ALL work is done.
type: feedback
---

Do NOT stop working to give status updates, ask for confirmation, or report progress mid-task. Just keep working until everything is complete.

**Why:** The user has repeatedly been frustrated by: "I've done X, should I continue?" or "X is done, here's what's next..." — these pauses waste time and make the user feel like they need to babysit. The user gave instructions — just execute them fully.

**How to apply:**
- When given a task, do ALL of it without stopping
- Don't commit and report after each sub-task — batch everything
- Don't ask "should I continue?" — the answer is always yes
- Don't list what's pending — just do it
- Only message the user when ALL work is truly complete
- If there are 50 files to update, update all 50 before saying anything
