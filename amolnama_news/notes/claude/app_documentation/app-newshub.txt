## Introduction
Main news collection hub — the largest app in the project.
Multi-step forms for 10+ news categories (extortion, crime, land grabbing,
price hike, sports, entertainment, war/conflict, July uprising, etc.).
Article publishing with sidenote layout, contributor management, engagement tracking.

## Features

### News Collection Forms
- 10+ form types: extortion, crime, land grabbing, price hike, sports, entertainment,
  war/conflict, July uprising, civic/community, watchdog, global news
- Multi-step wizard with stepper navigation
- Form persistence (localStorage auto-save between steps)
- Category → tag cascade (dependent dropdowns)
- Location cascade (division → district → upazila → union)
- Photo upload with asset_group_code (evidence, impact, accused, victim, witness, general)
- Edit pre-population from DB via helpers.py:build_edit_data()

### Article Display
- Two-column sidenote layout at /newshub/article/<slug>/
- Form type badge in meta bar
- Photo gallery grouped by asset_group_code
- Session-based likes, view count
- Shared actions-bar component

### Status Management
- Badge + pencil edit toggle → dropdown
- Owner/admin can change article status
- Inline messages, no popups

## Key Functions

### Views (views.py)
- news_collection_multistep_*() — 10+ form type handlers
- _build_common_extra() — shared context for all form types

### Helpers (helpers.py)
- build_edit_data() — reconstruct all form data from DB for edit pre-population
- build_article_seo_slug() — Bengali-safe slug via bangla_slugify()

## Troubleshooting
- CollNewsAsset composite PK — use raw SQL for file URLs (file_storage_path is computed column)
- Edit pre-population: news-form-edit-load.js sets hidden inputs BEFORE component scripts init
- Category → tag cascade: both _en and _bn in option text for Tom Select search
- hash_headline_check is computed column — remove from Django model, never include in INSERT
- Form persistence: localStorage key per form type, cleared on successful submit
