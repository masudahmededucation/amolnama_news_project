## Views

### home()
- Serves the pulse feed directly at /
- 3 tabs: For You, Following, My Posts (?tab= parameter)
- Uses build_post_feed_items() from post app
- Renders pulse/pages/pulse-home.html template
- Passes following_count for empty state messages

### article_detail()
- Article detail with JSON-LD NewsArticle schema
- SEO meta tags via seo context dict

### actions-bar
Like (session-based) + view count + share (Web Share API + clipboard) + edit link.
SVG share icon (X/Twitter style). JS flag instead of disabled attribute (cursor jump fix).
Init: window.actionsBar.init()

### photo-card
Image card with like/view/edit/delete/set-cover actions.
Init: window.photoCard.init()

### photo-lightbox
Full-screen image preview with arrow nav + keyboard navigation.
Init: window.photoLightbox.init()

## Utilities

### core/utils.py
- bangla_slugify(text) — NFC normalization (preserves Bengali matras). NEVER use Django slugify.
- generate_username_handle(display_name) — auto @username from display name, unique check.

## Base Template (core/base.html)
- HTML skeleton with nav, sidebar, main content, footer
- Loads all global CSS/JS: colors.css, base.css, navigation.css, header.css, etc.
- Loads Bengali input system: avro-phonetic.js → bangla-input.js → bangla-input-auto-attach.js → quill-avro.js → news-form-lang.js
- PWA: manifest link, service worker registration
- CSP headers for external CDNs (Quill, Tom Select, Cropper.js, Leaflet)

## Troubleshooting

### CSS :root hijacking
colors.css is the ONLY file with :root variables. App-specific CSS files (evaluation_vote, etc.)
had their own :root blocks that overwrote --primary. Always check for duplicate :root definitions
across all apps when colors misbehave.

### Global `a` color rule
Changed from `a { color: var(--accent) }` to `a { color: inherit }` + `main a { color: var(--accent) }`
to prevent blue bleed into header/nav/ticker.

### Sidebar not collapsing
width: 220px + min-width: 220px hardcoded. Fixed by removing min-width, adding
`.app-layout-collapsed .sidebar-navigation { width: 60px; }`.

### Static file caching
collectstatic --noinput after any static file edit. Dev server restart after settings changes.
Browser cache (Ctrl+Shift+Delete) is separate from server cache. But always check service worker first.

### Bengali slugify
NEVER use Django's slugify(allow_unicode=True) for Bengali. Always use bangla_slugify() from
core/utils.py. NFC normalization preserves matras unlike NFKD.

### Shared component init
All shared components require explicit init: window.actionsBar.init(), window.photoCard.init(),
window.photoLightbox.init(). Must be called after DOM ready in each page that uses them.
