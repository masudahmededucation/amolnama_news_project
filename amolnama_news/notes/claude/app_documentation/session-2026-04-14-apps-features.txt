## Features Built

### Twemoji Flag Rendering
- Windows doesn't render flag emojis natively
- Solution: _replace_flag_emoji_with_twemoji() in text_highlight.py
  replaces regional indicator Unicode pairs with Twemoji CDN SVG images
- Runs after entity detection + HTML build to avoid mention regex corruption
- Removed duplicate JS mention linkifier in post-feed-interactions.js
  that was corrupting img src attributes containing @14.0.2
- CDN: cdn.jsdelivr.net/gh/twitter/twemoji@14.0.2

### Shared Emoji Picker Component
- core/assets/js/components/emoji-picker.js + emoji-picker.css
- Used by: post composer (full mode, 6 categories), messenger (compact mode)
- Flag emojis rendered as Twemoji images + country code labels
- One source of truth — replaced 130 lines in post-home.js + 25 in messenger

### Watermark Remover Guide
- 3-step visual guide with sample images (step 1/2/3)
- Images compressed to 8KB each (400px width)
- Static assets in tools/static/tools/assets/img/watermark-guide/

### Post Composer Fixes
- Emoji picker: hidden attr toggle, draft mode guard, auto-close on textarea click
- Border-radius: --radius-full → --radius-lg (no text clipping)
- Mobile responsive: emoji grid 5 cols on <400px, media labels hidden

### Messenger Fixes
