EXECUTIVE SUMMARY
-----------------
Overall rating: 9.4 / 10 — enterprise-grade for asynchronous,
source-grounded, privacy-first quizzing in the Bengali market.

Beats commercial competitors on:
  - AI generation with NLI anti-hallucination
  - Source citation as a mandatory schema constraint
  - Webcam proctoring with ZERO IMAGES
  - Bengali / English bilingual first-class
  - Onboarding polish (workflow diagram + 11-FAQ help page)

Matches enterprise LMSes (Moodle / Canvas) on:
  - Quiz lifecycle (clone / archive / delete / schedule / version)
  - Bulk operations (import + export)
  - In-app notifications (pulse + messenger DM)
  - Webhook integrations
  - Custom certificates
  - Visual analytics charts

Below enterprise tier on:
  - Total question-type count (8 vs Moodle's 17)
  - QTI standards interop (skip — niche)

Remaining gaps are limited to:
  - 1 small admin UI gap (webhook subscription manager)
  - 1 small admin UI gap (accommodation override button)
  - 1 large blocker (no consumer app wired yet)

       Effort: small. Backend complete; need list + create + delete page.
       Last "API exists but no UI" gap.
  2. Accommodation override UI
       Effort: small. Backend complete; need a button on the proctoring
       session audit page.
  3. Consumer-app wiring (one consumer)
       Effort: medium. Mastermind has zero consumers; nothing renders
       quizzes to end users yet. Pick historybd / constitutionbd / quizadmin
       test-take page and wire end-to-end.

USEFUL BUT BIGGER BUILD:

       Reuse messenger schema. Engagement boost.
  5. 3 advanced question types
       calculated, drag-drop on image, regex.
       Niche — only if BCS / certified-exam users ask.
  6. Quiz template marketplace
       Clone-from-public. Network effect.

SKIP — BAD ROI FOR CURRENT MARKET:
  7. QTI import/export — only matters if migrating to/from Moodle/Canvas.
  8. Live multi-player quizzing — different product category (Kahoot tier).

     not relevant until multi-author teams exist.
