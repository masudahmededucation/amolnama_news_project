## Introduction
Travel Hub & Beauty of Bangladesh — community-driven destination guide
with photo/video galleries, reviews, accommodations, transport routes,
and travel tips. Two main sections: Travel Hub (destinations) and
Beauty Hub (photo/video gallery).

## Features

### Travel Hub
- Destination listing with district/category filters (Tom Select dropdowns)
- Add/edit destination with Quill.js rich text editor (Bengali typing via QuillAvro)
- Detail page with: description, photos, YouTube videos, reference links, accommodations, transport, tips, reviews
- Community contributions: anyone can add photos, YouTube links, reference links
- Edit/delete by: contribution owner + destination owner + staff/admin
- Leaflet map with Nominatim search for location (Dhaka-center default)
- Session-based likes on destinations, photos, videos

### Beauty Hub
- Photo/video gallery with category filters
- Upload with title, description, category tagging
- Like and comment system

### Shared Components Used
- actions-bar (like + view + share + edit)
- photo-lightbox (full-screen preview)

## Troubleshooting
- Tom Select on district/category: use element.tomselect.setValue(value, true) not .value=
- Quill editor: HTML sanitized server-side via _sanitize_html()
- YouTube: thumbnail + play button → new tab (no iframe due to restriction errors)
- Map search Enter key: must preventDefault() inside <form>
- Bengali slugs: use bangla_slugify() from core/utils.py
- Nominatim: state_district=district, county=upazila. Use extractNameBefore() for parsing
