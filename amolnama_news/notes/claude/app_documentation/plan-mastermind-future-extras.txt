1. ADVANCED QUESTION TYPES (3 types)
-------------------------------------
Status: NOT STARTED
Estimated effort: 2-3 days each
Priority: LOW — only if BCS / certified-exam users ask

a) calculated  — math expression with variable substitution
   - DB: new coll_question_calculated_var table (var_name, min, max, step)
   - Engine: substitute random var values per session, evaluate formula,
     match student's numeric answer within tolerance
   - Use case: math / physics / chemistry exam prep

b) drag-drop on image  — student places labels on an image hot-spot
   - DB: new coll_question_image_hotspot table (x_percent, y_percent,
     radius_percent, label_text)
   - Engine: collect (label_id, x, y) tuples; check each landed within radius
   - Use case: anatomy, geography, parts-of-a-machine

c) regex / pattern  — fill-blank but answer matched against a regex
   - DB: extend coll_question_option with option_match_pattern column
   - Engine: re-use _grade_fill_blank with re.fullmatch fallback
   - Use case: code / language quizzes


2. QUIZ TEMPLATE MARKETPLACE
----------------------------
Status: NOT STARTED
Estimated effort: 3-5 days
Priority: MEDIUM — network effect if users share quiz templates

Vision:
  Staff or quiz-creators can publish a quiz as a "template". Others
  can browse the catalogue and clone it as a starting point.

DB:
  - coll_quiz_template_listing
      (link_quiz_id, listed_by_user_profile_id, listing_status_code,
       template_category_code, template_description, clone_count, listed_at)
  - All cloning already works via the existing /api/exam/<id>/clone/ endpoint

UI:
  /quizadmin/quiz-templates/  — gallery, filter by topic / language,
                                  click "Clone to my drafts"
  Quiz form — new "Publish as template" toggle in the Status section

Auth:
  - Anyone with edit access on a quiz can publish it as template.
  - Cloning requires quiz creator permission.
