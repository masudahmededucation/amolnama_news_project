When the user requests a structural change like "use universal X everywhere" or "single source of truth", implement it completely across ALL forms in a single session. Do not:
- Do partial changes (e.g., update crime forms but skip WCV/July)
- Spread the work across multiple conversations
- Wait to be reminded repeatedly

The user had to ask for ~3 weeks before the universal Personal Identity pattern was applied to all forms (WCV victim, WCV accused, July martyr, crime repeaters). The pattern was clearly specified in form-checker.txt from the start. This caused unnecessary frustration.

Rule: When user says "everywhere" — audit every form that has the relevant fields, list them, and update them all in one go. Don't stop at the first form or the "easy" ones.
