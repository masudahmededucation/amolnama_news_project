display-hidden to hidden Attribute Migration — Rules & Plan
=============================================================
Created: 2026-04-05
Scope: 355 HTML templates + 373 JS instances + 42 CSS rules


PROBLEM
-------
.display-hidden { display: none } is a CSS class in forms.css (line 73).
Before CSS loads, the class does nothing — elements are VISIBLE.
Overlays, modals, hidden panels, toggle elements all flash on page load.


ROOT CAUSE
----------
CSS classes only work after CSS files load. HTML hidden attribute works
immediately — browser default [hidden] { display: none } is built-in.


PROVEN PATTERN (already in project)
------------------------------------
photo-lightbox.html: <div class="photo-lightbox" id="photo-lightbox" hidden>
photo-lightbox.css:  .photo-lightbox:not([hidden]) { display: flex; }
photo-lightbox.js:   lightboxElement.hidden = false; // show
                     lightboxElement.hidden = true;  // hide

post-feed-interactions.js: newPostsPill.hidden = true/false;
post-home.css: .post-feed-new-posts-pill:not([hidden]) { display: block; }


MIGRATION RULES
---------------
Rule 1: HTML — replace class="display-hidden" with hidden attribute
  Before: <div class="foo-bar display-hidden" id="my-element">
  After:  <div class="foo-bar" id="my-element" hidden>

Rule 2: CSS — add :not([hidden]) ONLY where element has explicit display rule
  Before: .foo-bar { display: flex; align-items: center; }
  After:  .foo-bar:not([hidden]) { display: flex; } .foo-bar { align-items: center; }
  OR:     .foo-bar:not([hidden]) { display: flex; align-items: center; }

  If element has NO explicit display rule, no CSS change needed.

Rule 3: JS — replace classList methods with .hidden property
  classList.add('display-hidden')           -> element.hidden = true
  classList.remove('display-hidden')        -> element.hidden = false
  classList.toggle('display-hidden')        -> element.hidden = !element.hidden
  classList.toggle('display-hidden', bool)  -> element.hidden = bool
  classList.contains('display-hidden')      -> element.hidden

Rule 4: Keep .display-hidden CSS rule as fallback until ALL files migrated
  Remove it ONLY after 100% verification.
